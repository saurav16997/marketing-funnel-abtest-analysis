# Marketing Funnel Analysis & A/B Testing Experiment

## Project Overview
This project simulates a real-world **growth analytics workflow** used by product and marketing analytics teams.  
Using large-scale eCommerce behavioral data, the project analyzes the customer acquisition funnel, identifies conversion drop-offs, and evaluates an A/B experiment to test potential improvements.

The analysis demonstrates how data analysts measure user behavior, diagnose funnel friction, and evaluate product experiments using statistical testing.

---

## Dataset
Source: Kaggle – eCommerce Behavior Dataset

The dataset contains **user interaction events from an online store**, including:

- Product views
- Add-to-cart actions
- Purchases
- User sessions

Each row represents an event generated by a user interacting with a product.

Example fields:

- `event_time`
- `event_type`
- `product_id`
- `category_code`
- `brand`
- `price`
- `user_id`
- `user_session`

## Funnel Analysis

User behavior was analyzed across the following funnel stages:
View → Cart → Purchase


### Session-Level Funnel Metrics

| Metric | Value |
|------|------|
View → Cart Conversion | ~4.1% |
Cart → Purchase Conversion | ~56.7% |
View → Purchase Conversion | ~6.8% |
Cart Abandonment Rate | ~43% |

### Key Insight

The **largest drop-off occurs between the View → Cart stage**, indicating friction in product engagement or purchase intent.

Potential improvements could include:

- Better product recommendations
- Improved product page UX
- Stronger call-to-action placement
- Social proof or urgency signals

---

## A/B Testing Simulation

To evaluate potential improvements, a simulated **A/B experiment** was conducted.

### Experiment Setup

| Variant | Description |
|------|------|
Variant A | Control experience |
Variant B | New experimental variation |

Metric measured: Conversion Rate (Purchase)


---

## Experiment Results

| Variant | Conversion Rate |
|------|------|
Variant A | 6.89% |
Variant B | 6.77% |

Statistical testing was performed using a **two-proportion z-test**.
z-score: -0.79
p-value: 0.425


### Interpretation

The difference between variants is **not statistically significant**.

Since the p-value is greater than 0.05, the observed difference is likely due to **random variation rather than a true effect**.

---

## Confidence Intervals

95% confidence intervals for conversion rates:
Variant A: 6.69% – 7.09%
Variant B: 6.57% – 6.97%


The intervals **overlap substantially**, further confirming the lack of a statistically meaningful difference.

---

## Business Conclusion

The experiment does **not provide sufficient evidence** that Variant B improves conversion.

Recommendation:Do not ship Variant B


Future experiments should focus on improving the **View → Cart conversion stage**, which represents the primary funnel bottleneck.
